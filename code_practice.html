Terminal

Today we will learn some basic commands to work your way around the terminal

How to enter the terminal:

  1. Hit Cmd+space and type in Terminal then push Enter.
  2. Now you are in the terminal.

Here is a list of commands useful in the terminal:

  • CD used to change your location
  • echo used to create new files
  • mkdir used to create new directories
  • cp used to copy files between directories
  • rm used to delete files
